# netwifi
NetWifi is minimal shell script for wireless connection. Designed for minimal window manager like Fluxbox, i3WM, or BSPWM.

# dependencies
1. wpa_supplicant
2. dhclient

# install & usage
```
git clone https://github.com/mahirrudin/netwifi.git
sudo cp netwifi /usr/local/bin/
sudo netwifi
```
